# 2016级项目实训成果展示
 

## 《游迹》 - 测试方向


### 项目简介

“游迹”是一款旅行记录和分享App，用户可以通过android设备便捷建立自己的旅行游记，图文并茂，App会记录时间和地理位置信息，可以和好友分享你的方位、游览景色、旅游心情等等。可以随时通过android设备，查找旅游目的地，查看其它用户的行程攻略作为参考，给自己的旅行提前做好准备和参考，沟通热爱旅游的小伙伴。
